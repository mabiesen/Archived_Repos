A Pi Server Story by Matthew Biesen
It works!

My name is Matthew Biesen and I created a pi-based webserver using information found at the following link GettingStarted!!

I found that getting started was incredibly easy, but despite my book learning I am finding that there is much more to learn. My first observation included some confusion over how to format the page: CSS script was overriding my HTML bold/italics/etc. I look forward to mastering this new environment.

Scott, if you have taken the time to read this web page, get on it bro. Just like VBA I would recommend an application based curriculum. Reading books did help me significantly, but I feel like the books should be read after getting started by hacking it.

What's Next?
